## 개요
* 성장 여력이 크나 타 산업 대비 낮은 온라인 침투율에 주안점을 두어, 현재 서비스 중인 마켓컬리를 벤치마킹해 식자재를 판매하는 웹사이트를 개발했습니다.
* 인원 : 4명
* 개발 기간 : 2022.08.03 ~ 2022.08.17 (14일)

---
## 맡은 기능
### 1. 회원가입, 로그인, 로그아웃
  - 회원가입시 이메일 인증
  - 소셜 회원가입, 로그인 (카카오, 네이버)
### 2. 장바구니
  - CRUD(생성, 조회, 갱신, 삭제)
### 3. 찜한상품
  - CRUD(생성, 조회, 갱신, 삭제)
### 4. 게시판
  - 조회, 삭제
  - 페이지네이션
### 5. 개인정보 수정
